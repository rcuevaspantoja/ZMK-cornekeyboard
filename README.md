# ZMK corne keyboard with custom case
This keyboard layout is made with Spanish / English in mind and confortable to use while coding with easy access to keys and symbols, you have to select the following keyboard layout on Windows to be able to create characters like ñ,é,á with RALT on the bottom left side and the key like this RALT + n => ñ and so on.
